# 📖 3D Book Portfolio — Guilherme Eller Coan

Um portfólio interativo com animação de livro 3D, desenvolvido com HTML, CSS e JavaScript puro.

🔗 **[Ver online](https://guidevin.github.io/)**

---

## ✨ Funcionalidades

- 📚 Animação de livro 3D com virada de páginas
- 🖱️ Navegação por clique nos botões de próxima/anterior página
- ⌨️ Navegação pelo teclado com as setas `←` e `→`
- 🎬 Animação de abertura automática ao carregar a página
- 📱 Layout responsivo

---

## 🗂️ Páginas do Livro

| Página | Conteúdo |
|--------|----------|
| Capa | Perfil, redes sociais e botão de contato |
| 1 | Experiência de Trabalho |
| 2 | Educação |
| 3 | Meus Serviços |
| 4 | Minhas Habilidades |
| 5 | Último Projeto |
| 6 | Formulário de Contato |

---

## ⌨️ Atalhos de Teclado

| Tecla | Ação |
|-------|------|
| `→` ArrowRight | Próxima página |
| `←` ArrowLeft | Página anterior |

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3 (animações 3D com `transform` e `perspective`)
- JavaScript (Vanilla)
- [Boxicons](https://boxicons.com/) — ícones
